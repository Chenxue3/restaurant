## Chatbot Feature

### Description
The chatbot is an AI-powered assistant integrated into the application to provide users with quick and interactive responses to their queries. It is designed to answer questions related to food and restaurants.

### Features
- **Default Questions**: Users can select from a list of predefined questions for quick answers.
- **Custom Queries**: Users can type their own questions to receive personalized responses.
- **Interactive Responses**: The chatbot provides formatted and structured answers for better readability.
- **Real-Time Updates**: The chatbot scrolls to the latest message automatically.
